# AI Ticketing System

## 📌 Overview
This is a full-stack AI-based ticketing system that allows users to submit issues and automatically analyzes them. The system classifies tickets based on category, severity, and department, and provides an appropriate response.

## 🎯 Objective
To build a smart support system that reduces manual effort by automatically handling and routing support tickets.

## ⚙️ Features
- Ticket submission interface
- Automatic issue classification
- Department routing (IT, Engineering, Finance, etc.)
- Auto-response generation
- Real-time frontend-backend interaction

## 🛠️ Tech Stack
- Frontend: React.js
- Backend: FastAPI (Python)
- API Communication: Axios

## 🚀 How It Works
1. User enters issue in frontend
2. Frontend sends request to backend API
3. Backend processes the request using logic
4. Response is sent back and displayed to user
